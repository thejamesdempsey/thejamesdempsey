SmartClickr

SmartClickr, a project I built with 3 classmates this past year, is an all-in-one audience response system build to help facilitate better communication with a speaker and audience members. It is a web app built to run on any internet connected device. It is super easy and fun to use and it is built on Node.js so it is lighting quick. The responses from the audience grow on charts in real-time. You should really check it out, it will rock your socks!
